# Image Resizing
Image Resizing using these methods:
* Nearest Neighbor
* Bilinear Lagrange Interpolation of multiple grades
* Average between 2 images

## Dependencies
* libjpeg
* libpng

## Run
```
git clone git@github.com:omarsalazars/BicubicInterpolation.git
cd BicubicInterpolation
make
./interpolation
```
